# 爵士、融合与即兴音乐

## 核心顺序

1. 锁定录音版本、阵容、日期和编制；同一标准曲的结构、调性、速度与和声可能因版本而异。
2. 标记 intro、head in、solo choruses、interlude、trading、shout chorus、head out 与 coda；逐 chorus 记录变化。
3. 区分 lead sheet、编曲和实际演奏；不要把谱面和弦当作完整 voicing，也不要忽略再和声与替代。
4. 同时分析 time feel、comping、bass motion、鼓组互动、solo arc 与集体能量。

## 子流派入口

| 分支 | 优先检查 |
|---|---|
| New Orleans / traditional jazz | 集体即兴、front line 角色、二拍/四拍、break、ragtime/blues 语汇 |
| swing / big band | section voicing、riff、背景乐句、soli、shout chorus、四拍 walking 与舞蹈功能 |
| bebop | 快速功能和声、contrafact、enclosure、chromatic approach、目标音与连续八分音符 |
| hard bop | blues/gospel、riff head、强 backbeat、soulful articulation 与更具重量的 rhythm section |
| cool / West Coast | 轻触发音、线性对位、空间、编配音色与克制动态；不要等同于慢速 |
| modal jazz | 长时和声、调式中心、pedal、motivic development、音色与密度推进 |
| soul jazz / organ jazz | gospel/blues、vamp、organ registration、groove、backbeat 与 call-response |
| post-bop | 功能/调式混合、复杂拍感、开放 voicing、主题碎片与高互动性 |
| free / avant-garde | 实时规则、音色、注册、事件密度、集体协商与形式涌现；不强套 chorus form |
| third stream / chamber jazz | 古典曲式/配器与爵士即兴的角色分配、记谱/自由边界、室内乐平衡 |
| fusion / jazz-rock | 电声乐器、rock/funk groove、合成器、延长 solo、复杂拍号与制作音色 |
| Afro-Cuban jazz | clave 方向、tumbao、cáscara、montuno、打击乐角色与 jazz harmony 的结合 |
| Brazilian jazz / bossa / samba jazz | samba/partido-alto 重音组织、surdo/吉他型、切分、和声延伸与葡语韵律；勿与 Afro-Cuban 混同 |
| vocal jazz | 歌词处理、behind/ahead phrasing、melodic variation、scat、伴奏呼吸与音色 |
| contemporary / nu jazz | hip-hop、broken beat、电子制作、循环与即兴的比例；按需要加载其他现代模块 |
| solo / duo / chamber jazz | 隐含低音、内声、空间、角色交换、rubato 与即时编曲 |

## 和声与 voicing

- 追踪 guide tones、voice leading、ii–V、turnaround、secondary dominant、tritone substitution、backdoor、diminished passing、upper structure 与 slash bass。
- 延伸音是和弦身份、旋律音或 voicing 色彩，不默认需要按古典不协和规则解决。
- 标出 pianist/guitarist 的 comping rhythm、rootless voicing、quartal voicing、drop voicing、register 与留白。
- 在 modal、free 或 pedal-based 音乐中优先分析中心音、音阶材料、音程集合与音色，而非制造虚假功能进行。

## 即兴与互动

- 把 solo 分成建立、发展、峰值、释放等阶段，追踪动机重复、变形、序进、节奏置换、引用和音域扩张。
- 说明 soloist 如何落在 chord tones、extensions、approach notes 或 outside material，以及何时重新进入内部语汇。
- 记录鼓手的 comping、setup、metric displacement，贝司的 walking/ostinato/pedal，和声乐器的回应与空间让渡。
- 区分 behind/ahead of beat 与真正速度变化；描述微时值关系而非只写“很有 swing”。

## 大乐队与编曲

- 分析 sax/brass/rhythm sections 的角色、unison/soli、block/drop voicing、call-response、背景乐句和 shout chorus。
- 检查 lead trumpet、低音乐器、加倍、音区冲突、动态平衡及 solo 与 ensemble 的交接。

## 常见误判

- 不因出现七和弦就称为 jazz，也不因没有复杂和弦就否定其爵士属性。
- 不把 swing ratio 当作固定数值；它随速度、乐手、时期和段落变化。
- 不把即兴描述成无结构，也不把转录出的音符列表当作即兴分析。
- 拉丁爵士必须识别具体节奏与文化来源，避免用笼统“Latin feel”覆盖不同传统。
