# 输入、证据与来源

## 证据层级

在分析笔记和成稿中区分四类陈述：

1. **来源事实**：出版信息、作曲年份、首演、编制、作曲家言论等；附可靠来源。
2. **直接观察**：可从所用谱本、音频或视频定位的内容；附小节号、排练号或时间戳。
3. **分析解释**：对观察的理论归纳，例如把一组段落解释为回旋曲或把和弦解释为增六和弦；说明依据。
4. **推断/假说**：关于意图、影响、象征或听觉歧义的合理猜测；使用限定语并给替代解释。

不要把类别 3 或 4 写成类别 1。作曲家意图尤其需要手稿、书信、访谈或可靠研究支持。

## 按输入类型工作

### 总谱或分谱

- 记录版本、编辑者、出版者、页码、小节编号规则和移调乐器。
- 检查弱起、反复、第一/第二结尾、D.C./D.S./Coda、删节和异稿。
- 只有分谱时避免断言完整和声与整体配器。

### 音频或视频

- 记录录音/演出版本、时长、速度与关键时间戳。
- 至少进行三轮聆听：宏观结构、局部材料、综合复核。
- 把演奏或制作特征与作品本体分开；现场视频中的视觉线索可辅助识别乐器，但需谨慎。

### MIDI 或 MusicXML

- 先检查速度、拍号、调号、力度、踏板、声部和音色映射是否可信。
- MIDI 音色不等于真实配器；量化数据也不必然代表演奏微时值。
- MusicXML 可能丢失排版、奏法或特殊记谱，必要时与渲染谱面核对。

### 和弦谱、lead sheet 或歌词

- 把它视为简化模型，不假设其包含转位、内声部、装饰和替代和弦。
- 标注和弦谱来源与版本；流行歌曲的现场版、降调版和翻唱版常不同。
- 只有歌词时可以分析文本结构与韵律，但不能凭歌词断言音乐结构。

### 仅有曲名

- 先确认同名作品、乐章、版本、表演者与链接。
- 若无法获得谱面或音频，只做有来源的背景与已有研究综述，并明确不能完成的听觉/谱面分析。

## 来源优先级

按具体主张选择最接近原始证据的来源：

1. 作曲家手稿、书信、日记、访谈、项目文件、官方作品目录。
2. 权威乐谱版本、出版社说明、档案馆、委约机构、首演单位。
3. 同行评审论文、学术专著、学位论文、专题目录与可靠传记。
4. 博物馆、大学、专业乐团、歌剧院、唱片公司制作说明。
5. 高质量访谈、专业评论和教育资源。

百科、博客、乐迷网站和自动和弦网站适合发现线索，不应单独支撑关键争议性结论。对网络资料记录作者、标题、机构、发布日期和访问链接。背景信息若存在学术分歧，呈现主要观点及证据，不伪造共识。

## 版本与可复现性

在成稿开头写明：

- 作品与乐章/歌曲名
- 作曲者、编曲者、表演者（相关时）
- 乐谱版次或录音/视频链接
- 分析定位系统：小节号、排练号或时间戳
- 转调或移调处理方式

版本不明时，将结论限定为“就所提供录音/谱本而言”。
