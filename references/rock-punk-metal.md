# 摇滚、朋克、金属及其衍生风格

## 核心顺序

1. 锁定调弦、音高中心、速度、拍感和主要 riff；区分吉他实际音高、指法位置与移调乐器记法。
2. 把 riff 当作音高、节奏、奏法、音色和鼓/贝司关系的复合单元，不只还原和弦名称。
3. 画出 intro、verse、chorus、pre-chorus、bridge、solo、breakdown、interlude、coda，并记录动态与失真密度。
4. 分开分析 songwriting、乐队演奏、吉他层叠、录音编辑和混音响度。

## 子流派入口

| 分支 | 优先检查 |
|---|---|
| blues rock / classic rock | blues 语汇、shuffle/straight、riff、五声音阶、call-response、吉他 solo 与松紧拍感 |
| hard rock / arena rock | power chord、anthemic chorus、riff/歌唱对比、鼓组规模、吉他叠录与动态高潮 |
| punk / hardcore | 高速 downstroke、短曲式、群唱、break、DIY 音色、现场能量；复杂度低不等于结构弱 |
| post-punk / goth | ostinato bass、棱角吉他、鼓机/重复、负空间、效果器与冷色制作 |
| indie / garage rock | 松散微时值、音色个性、简化 riff、房间感、表演不完美与段落对比 |
| grunge / alternative rock | quiet-loud、降调弦、失真层、旋律/噪声反差、沉重 backbeat 与人声质地 |
| psychedelic / space rock | 调式 jam、持续音、相位/延迟、录音室处理、扩展曲式与感知空间 |
| shoegaze | 多层吉他、fuzz/reverb/modulation、频谱融合、人声后置、和声模糊与音墙动态 |
| post-rock | 长程 build、动机累积、器乐叙事、音色渐变、延迟织体与高潮密度 |
| progressive / math rock | 不规则拍、metric modulation、长曲式、主题回归、技术段落的功能、吉他/鼓互锁 |
| heavy / speed / power metal | riff 与主旋律双中心、双吉他、快速低音、双踩、史诗/叙事段落与调式语汇 |
| thrash / death / black metal | palm mute、tremolo picking、blast beat、riff 模块、极端声线、调式/半音中心与制作美学 |
| doom / sludge / stoner metal | 慢速重量、低调弦、持续失真、riff 留白、拖后拍感、低频与长时值结构 |
| metalcore / deathcore | breakdown、syncopated chug、双踩、clean/harsh 对比、编辑精度与段落冲击 |
| nu metal / rap metal | 低调弦 groove、切分 riff、rap/clean/harsh 分工、采样/turntable 与 funk/hip-hop 关系 |
| industrial rock / metal | sequencer、机器律动、采样、重复 riff、噪声、声像与电子层；同时读取电子模块 |
| djent / progressive metal | 扩展音域吉他、低音高切分、polymeter 感、机械紧密度、音色门限与复杂曲式 |
| emo / pop-punk | 告白式文本、octave line、动态释放、群唱 hook、半时 breakdown 与青春叙事 |

## Riff、和声与调弦

- 标出 riff 的核心音程、重音网格、休止、重复单元、尾部变体和与鼓点的同步方式。
- 区分 power chord 的根音运动与完整功能和声；无三音并不自动意味着大小调不明，需结合旋律和中心音。
- 检查 pedal tone、modal mixture、chromatic planing、tritone、降二级、降六级、五声音阶和 blues inflection。
- 降调弦会改变音域、指法与重量感，不必然改变调性；说明 drop tuning 如何便利特定 riff 与和弦形状。

## 乐队与制作

- 吉他：区分左右声道双轨、同度/八度加倍、clean/crunch/high-gain、pickup/amp/cab、fuzz、delay、reverb、modulation 与 feedback；不猜具体型号。
- 贝司：判断是跟随根音、加倍 riff、独立对位还是承担旋律；检查拨片/指弹/失真与 kick 的低频分工。
- 鼓：分析 backbeat、ghost note、fill、double-kick、blast beat、half-time、ride/hat 变化及是否锁定吉他切分。
- 人声：区分 clean、belt、shout、scream、growl、gang vocal、double 与空间处理，说明其段落和叙事作用。
- 制作：比较房间感与网格编辑、动态余量与限制器、真实叠录与 re-amp/采样增强；不要用“原始/精致”代替机制。

## 常见误判

- 不因速度快就判为 punk/thrash，也不因低调弦和 growl 就判为某一金属分支。
- 不把所有失真声部合并成一把吉他；检查声道、八度、和声、反馈与噪声层。
- 不以技术复杂度作为价值排序；分析复杂或简约手段各自怎样服务风格与表达。
- 不忽视现场版和录音室版在速度、调弦、solo、编制、编辑与音墙密度上的差异。
