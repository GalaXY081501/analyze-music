# 流行、R&B、嘻哈及相关分支

## 核心顺序

1. 画出 verse、pre-chorus、chorus、post-chorus、bridge、break、rap verse、dance break 与 outro，并记录 hook 出现方式。
2. 分开分析 songwriting、arrangement、performance 与 production；不要把制作变化误写成新作曲段落，也不要忽略制作本身的曲式功能。
3. 追踪鼓/打击乐、bass/808、和声乐器、主唱、和声、ad-lib、采样与效果层的进入退出。
4. 解释歌词叙事、韵脚、语言重音、flow、旋律音域和声线变化如何共同建立身份与高潮。

## 子流派入口

| 分支 | 优先检查 |
|---|---|
| 主流 pop / dance-pop | hook 密度、pre-chorus lift、chorus payoff、四拍律动、vocal stack、段落增减法 |
| singer-songwriter / folk-pop | 叙事视角、语言韵律、和声节奏、原声伴奏型、演唱细节、编曲扩张 |
| J-Pop / anisong | 高信息量旋律、频繁段落、转调/副属和弦、乐器层叠、动漫/媒介叙事功能 |
| K-Pop | 多段拼接、风格切换、rap/vocal 分工、舞蹈段、音色钩子、表演与影像协同 |
| City Pop / AOR / yacht rock | maj7/9、secondary dominant、line cliché、电钢琴/吉他 comping、旋律贝司与精密鼓组 |
| synth-pop / new wave | sequencer、synth bass、鼓机、门限混响、音色角色、机械网格与人声反差 |
| indie pop / dream pop / bedroom pop | lo-fi 痕迹、混响景深、吉他/合成器雾化、弱化终止、亲密人声与制作限制的美学化 |
| R&B / contemporary R&B | pocket、melisma、延伸和弦、guide-tone line、sub-bass、和声层与节拍留白 |
| neo-soul | behind-the-beat、非方整和声节奏、rootless/upper-structure voicing、微时值、声部滑动 |
| funk / disco | “一拍”重心、切分互锁、短促和弦、ghost note、八度贝司、vamp 与持续舞动功能 |
| boom bap / sample-based hip-hop | break、sample chop、loop seam、鼓点摆动、flow 与采样再语境化；核对采样来源 |
| trap / drill | 半拍感、808 音高与滑音、hi-hat roll/triplet、稀疏和声、flow 网格与低频空间 |
| hyperpop / experimental pop | form fragmentation、极端音高/失真、剪辑、clipping 美学、vocal transform 与强烈段落反差 |
| reggae / dub / dancehall | skank、one-drop/steppers、重低音、空间效果、版本文化、riddim 与 sound-system 语境 |
| Latin pop / reggaeton | dembow、clave 关系、montuno/吉他型、打击乐分层；不要混同不同拉丁与加勒比传统 |
| Afrobeats / amapiano 等 | 多层打击乐、跨拍重音、bass/log drum、循环微变与地域场景；使用具体来源避免泛化“非洲节奏” |

## 和声与旋律

- 先判断循环是否承担稳定平台、方向推进或段落区分；四和弦循环不等于和声简单。
- 用音阶级数、调式中心、bass line、slash chord、modal interchange、secondary dominant 和 chromatic approach 描述实际机制。
- hook 可来自动机、节奏、歌词、音色或制作手势，不必等同于副歌旋律。
- 比较主歌与副歌的音域、重音、元音长度、旋律密度和重复率，而不只说“副歌更高”。

## Groove、演唱与制作

- 记录 kick、snare/clap、hat、percussion 与 bass 的互锁位置；区分 straight、swing、shuffle、laid-back 和 pushed。
- 分析主唱的胸声/混声/假声、气声、咬字、滑音、颤音、rap cadence、double、harmony、ad-lib 与自动调音的表达作用。
- 追踪 mute、filter、riser、impact、dropout、vocal chop、reverse、automation、sidechain、distortion 和 stereo width 如何标记段落。
- 区分创作者、制作人、编曲者、演奏者、采样来源和混音贡献；证据不足时不武断归因。

## 常见误判

- 不把重复解释为缺乏发展；检查每轮的层次、节奏、歌词、音色与能量微变。
- 不因使用 maj7 就称为 jazz，也不因 808 就自动归为 trap。
- 不把所有非英语流行音乐套入欧美模板；考虑本地歌谣、语言、媒介产业与表演传统。
- 不把响度、修音或量化直接等同于缺乏技巧；分析它们承担的审美与结构功能。
