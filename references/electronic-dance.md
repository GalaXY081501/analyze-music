# 电子音乐、舞曲与声音设计

## 核心顺序

1. 确定 BPM、拍感、网格、swing、主要 drum pattern、低频系统和 loop 长度；必要时区分半拍/双倍速度感。
2. 以 8/16/32 小节为单位画出 DJ intro、groove、build、drop、breakdown、re-entry、outro 与过渡。
3. 分开记录作曲音高、节奏编程、合成/采样、自动化、效果、混音和母带，不把所有变化统称为“编曲”。
4. 追踪 kick、sub/bass、snare/clap、hat/shaker、percussion、lead、pad、arp、vocal/sample 和 FX 的频谱与空间角色。

## 子流派入口

| 分支 | 优先检查 |
|---|---|
| house：deep / soulful / piano | four-on-the-floor、offbeat hat、clap、bass groove、vocal/piano hook、滤波与温暖空间 |
| house：tech / progressive / acid | 极简 loop、渐进自动化、303 式共振滤波、长程 build 与 DJ 混接功能 |
| techno：Detroit / minimal / dub | 机器 funk、重复微变、polyrhythmic percussion、delay space、纹理与长程张力 |
| techno：industrial / hard | 失真 kick、噪声层、金属质地、压缩密度、强瞬态与黑暗频谱 |
| trance：uplifting / progressive | supersaw、长 build、breakdown、和声释放、arp、snare roll 与大尺度高潮 |
| psytrance | 滚动 bass、kick-bass 交替、短促音色、快速自动化、迷幻空间与高事件率 |
| breakbeat / UK breaks | 切分 break、重排鼓片、ghost hit、sub 与非四拍重心 |
| jungle / drum & bass | amen/break chop、双倍速度鼓、半拍低频、reese/sub、drop 对比与快速编辑 |
| UK garage / 2-step | 缺拍 kick、shuffle hats、切碎人声、弹性 sub 与 swing 网格 |
| dubstep / bass music | half-time、sub movement、wobble/modulation、空间留白、drop sound design 与 call-response bass |
| future bass / melodic bass | sidechain swell、宽和弦、vocal chop、half-time、pitch modulation 与旋律化 drop |
| electro / breakdance | 808、syncopated break、robotic funk、vocoder 与序列器 |
| footwork / juke | 高速网格、非对称 kick、切碎人声、重复短样本、低频重音与舞蹈功能 |
| ambient / drone | 缓慢演化、频谱、包络、持续音、空间、噪声底与注意力时间；不强求 beat/drop |
| IDM / glitch | 非规则编辑、微切片、复杂网格、生成过程、音色事件与可感知结构的关系 |
| downtempo / trip-hop | 慢速 break、采样尘埃感、重低音、电影化和声与亲密/疏离人声 |
| synthwave / vaporwave | 复古合成器、鼓机、门限混响、磁带/采样降质、怀旧互文与再语境化 |
| hardstyle / hard dance | pitched kick、reverse bass、build/drop、旋律主题与 kick 音色演化 |
| hardcore / gabber | 高速失真 kick、rave stab、极端压缩、break 与持续强度管理 |
| experimental / noise electronic | 反馈、失真、频谱块、过程、空间和事件密度；以声音组织而非传统和弦评价 |

## 节奏与低频

- 写出 kick/snare/hat 的实际落点和 ghost notes；区分 straight 16ths、shuffle、triplet grid、microtiming 与随机 humanization。
- 检查 kick 与 sub 是交替、重叠、sidechain ducking 还是频段分工；不要仅凭听感猜具体压缩器设置。
- 说明 bass 的音高、包络、滑音、失真谐波与 mono/stereo 管理如何影响 groove 和可播放性。
- break-based 音乐需分析切片边界、重排、time-stretch、pitch 与原始 break 的再语境化；核对可验证的采样来源。

## 合成、采样与自动化

- 按声音功能描述 oscillator/sample source、filter、amplitude/filter envelope、LFO/modulation、distortion、delay/reverb 与 resampling。
- 无可靠工程文件时不要断言具体合成器、插件、预设或精确参数；给出可能的实现路径和置信度。
- 追踪 cutoff、resonance、envelope、wavetable position、pitch、send、stereo width、density 与 silence 的自动化曲线。
- 把 riser、downlifter、impact、reverse、fill、stop、vocal chop 和 noise sweep 视作句法标点，说明其位置与功能。

## 曲式、空间与母带

- 舞曲的重复应按每轮增减、频谱开放、fill、automation、低频切换和混接功能分析。
- 比较 breakdown 与 drop 的响度、低频、瞬态、声场宽度和事件率；避免只说“更炸”。
- 区分编曲密度、混音压缩和母带响度；只有成品音频时不声称知道精确 LUFS、动态处理链或 stem 结构。
- 评价空间时区分 early reflection、reverb tail、pre-delay、tempo-synced delay、mono compatibility 与前后景。

## 常见误判

- 不以 BPM 单独决定 house、techno、trance 或 drum & bass；结合鼓型、低频、音色、曲式和场景。
- 不把所有四拍电子乐称为 EDM，也不把所有无鼓电子音乐称为 ambient。
- 不因和声循环静态就判定缺乏发展；电子音乐的主要形式参数可能是音色、频谱、空间与事件率。
- 不把可听失真自动解释为削波错误；先判断它是 sound design、混音饱和还是实际技术问题。
