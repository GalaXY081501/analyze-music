# 分析维度与诊断问题

按材料和任务选择维度，不要求每次全部使用。

## 曲式与戏剧进程

- 边界由什么建立：终止、调性、主题、音色、密度、歌词、速度还是剪辑？
- 段落之间是重复、对比、展开、插入、过渡、回归还是尾声？
- 存在多层结构吗：乐句、乐段、章节、整曲？
- 期待在哪里建立、推迟、误导和满足？高潮与最低能量点在哪里？
- 奏鸣、回旋、变奏、二/三部、十二小节布鲁斯、AABA、verse–chorus 等标签是否真的解释材料？必要时采用中性段落标签。

## 旋律、主题与动机

- 核心身份来自音高、音程、轮廓、节奏、音色还是发音？
- 乐句长度规则还是不规则？问答、延伸、截断和重叠如何组织？
- 音域、最高/最低音和重音位置怎样塑造高潮？
- 动机通过何种变形跨越段落或声部？伴奏/低音是否也携带主题材料？

## 调性、调式、和声与对位

- 主音/中心音如何被建立？调号是否等于实际中心？
- 和声节奏在何处加快或减慢？这与结构有什么关系？
- 终止类型、转调路径、枢纽和弦与共同音如何组织方向？
- 低音是否形成独立线性逻辑、固定音型、持续音或半音线？
- 非和弦音属于经过、辅助、挂留、倚音、先现还是不可还原的色彩音？
- 声部是否存在模仿、卡农、赋格性进入、反向进行、平行或声部交叉？
- 爵士/流行中考虑转位、slash chord、upper structure、guide tones、tritone substitution、backdoor、modal interchange 与 line cliché。
- 后调性音乐可分析音级集合、轴心音、音程循环、对称结构、音块与音色和声；不要硬套功能和声。

## 节拍、节奏与律动

- 表层拍号与实际感知的拍群是否一致？有无复合拍、混合拍、加法节奏或换拍？
- 切分发生在哪一层：细分、拍、半小节、乐句？
- 是否存在 hemiola、cross-rhythm、polyrhythm、metric modulation、phase 或节拍歧义？
- groove 的骨架由谁承担？kick、snare/clap、hi-hat、bass 与和声乐器怎样咬合？
- 乐句是否提前/延后进入，形成 pickup、push 或拖后感？
- 节奏密度和休止如何调节能量与空间？

## 织体、配器与音色

- 当前是单音、同音、主调、复调、异音还是层状织体？是否混合？
- 旋律、低音、内声、节奏、填充与效果分别由谁承担？角色何时交换？
- 加倍发生在同度、八度或其他音程？是为增强、融合、扩宽还是突出？
- 音区、密度、发音与动态如何影响可听度和频谱空间？
- 木管/铜管/弦乐/人声/合成器/吉他等音色如何融合或对抗？
- 特殊奏法、静音器、弓法、气声、失真、滤波、采样或声学空间有什么结构功能？

## 力度、发音与表演实践

- 力度是局部表情还是长程曲线？是否与配器密度同步？
- 连断、重音、装饰、rubato、portamento、vibrato、swing 与音准处理如何改变句法？
- 作品所处时期/风格对速度、装饰、编制、调律或即兴有什么惯例？
- 不同演绎的差异来自谱本、传统、个人选择还是录音技术？

## 文本、叙事与媒介

- 音乐是否顺应语言重音和句法？是否故意错位？
- melisma、反复、停顿、升降调、和声冲突或音色变化突出哪些词？
- 歌曲中的 verse、pre-chorus、chorus、bridge 与歌词叙事阶段如何对应？
- 影视/游戏音乐与画面、剪辑、角色、空间、交互状态如何同步或反衬？区分 diegetic/non-diegetic 与主题变体。

## 录音、制作与空间

- 声像、景深、混响、延迟和声场宽度如何分层？
- 压缩、饱和、失真、自动化、滤波和频谱留白如何塑造段落？
- 采样、loop、编辑、drop、build、breakdown 与 transition effect 是否承担曲式功能？
- 区分作曲、编曲、演奏、声音设计、混音与母带层面的贡献；证据不足时不要武断归因。

## 风格、互文与接受

- 风格标签由哪些具体音乐特征支持？哪些特征与标签冲突？
- 是否存在引用、典故、topic、惯用和声、节奏、音色或制作手法？
- 同时代听众与今天的听众可能如何不同地理解这些材料？
- 作品在作曲家生涯、类型史或媒介史中处于什么位置？

## 综合原则

优先寻找跨维度关系，例如：

- 动机压缩 + 和声节奏加快 + 配器增厚共同制造高潮。
- 共同音持续 + 低音下行 + 音色逐步变暗形成表面静止、深层移动。
- 副歌旋律音域上移 + 元音拉长 + backbeat 加强共同扩大宣泄感。

综合段落必须回答“这些机制共同造成了什么”，不能只复述各章节。
